# Beatles theme

An 8-theme VS Code color theme pack inspired by the Beatles. Dark and light variants for each Beatle.

## Themes

### Beatles: John Lennon
Olive drab, stark white, and charcoal.

### Beatles: Paul McCartney
Sky blue, sunny yellow, and deep mahogany.

### Beatles: George Harrison
Saffron orange, royal purple, and forest green.

### Beatles: Ringo Starr
Cherry red, electric blue, and silver sparkle.

Each theme comes in a **dark** and **light** variant.

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Beatles theme"
4. Click Install
5. Open the Command Palette (Ctrl+K Ctrl+T) and select your favourite Beatle
