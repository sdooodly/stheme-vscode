# Beatles Theme

A 4-theme VS Code color theme pack inspired by the Beatles.

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Beatles Theme"
4. Click Install
5. Open the Command Palette (Ctrl+K Ctrl+T) and select your favourite Beatle

## Color Palettes

### John
- Background: `#0A0A0A`
- Accent: `#FF4500` (fire red)
- Strings: `#FF6B35` (orange)
- Keywords: `#FFFFFF` (stark white)

### Paul
- Background: `#1A2035`
- Accent: `#D4A843` (honey gold)
- Strings: `#D4A843` (warm gold)
- Keywords: `#7AABD4` (soft blue)

### George
- Background: `#1A1230`
- Accent: `#E89B2D` (saffron)
- Strings: `#E89B2D` (saffron orange)
- Keywords: `#A87FD4` (purple)

### Ringo
- Background: `#0F1E3D`
- Accent: `#FFD700` (bright yellow)
- Strings: `#7DCE82` (playful green)
- Keywords: `#5CB8E6` (sky blue)
